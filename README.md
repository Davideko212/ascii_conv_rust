# ascii_conv_rust
Rust CLI App that converts images to ASCII art

## Arguments

---
### Required
- Input path
> `--input-path <PATH>` or `-i <PATH>`
- Output path
> `--output-path <PATH>` or `-o <PATH>`

### Optional
- Resolution
> `--resolution <WIDTHxHEIGHT>` or `-r <WIDTHxHEIGHT>`
- Big Charset
> `--big-charset` or `-b`
- Time
> `--time` or `-t`
- Preview
> `--preview` or `-p`
